Veilarbregistrering
================

Applikasjon for nyregistrering av arbeidssøkere.

# Komme i gang

```
# bygge
mvn clean install 

# test
mvn test

# starte
# Kjør main-metoden i Main.java
# For lokal test kjøring kjør MainTest.java
```
